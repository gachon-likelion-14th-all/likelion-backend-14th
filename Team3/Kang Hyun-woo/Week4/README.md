4주차 실습 과제 (자료구조 + 함수)

----------------------------------------
문제 1. 학생 점수 관리 프로그램 (배열)

[설명]
학생 수를 입력받고, 각 학생의 점수를 입력받아 전체 점수, 평균, 최고 점수를 출력하는 프로그램을 작성하시오.

[요구사항]
1. 학생 수 입력
2. 배열을 이용하여 점수 저장
3. 전체 점수 출력
4. 평균 점수 출력
5. 최고 점수 출력

[조건]
- 배열(int[]) 사용 필수
- 반복문(for) 사용
- 평균은 소수점까지 출력

----------------------------------------
문제 2. 단어 빈도수 분석 (Map)

[설명]
문장을 입력받아 각 단어가 몇 번 등장했는지 출력하는 프로그램을 작성하시오.

[요구사항]
1. 문자열 입력
2. 공백 기준으로 단어 분리
3. Map을 이용하여 단어 개수 저장
4. 결과 출력

[조건]
- HashMap 사용
- split() 활용
- 반복문 사용

----------------------------------------
문제 3. 계산 함수 만들기 (함수)

[설명]
두 개의 정수를 입력받아 사칙연산을 각각 함수로 만들어 계산하는 프로그램을 작성하시오.

[요구사항]
1. 두 개의 정수 입력
2. 아래 함수 구현
    - 덧셈
    - 뺄셈
    - 곱셈
    - 나눗셈
3. 결과 출력

[조건]
- 함수는 main 밖에 작성
- return 사용

----------------------------------------
문제 4. 짝수/홀수 판별 함수

[설명]
정수를 입력받아 짝수인지 홀수인지 판별하는 함수를 작성하시오.

[요구사항]
1. 정수 입력
2. 짝수 판별 함수 작성
3. 결과 출력

[조건]
- boolean 반환
- main에서 결과 출력
