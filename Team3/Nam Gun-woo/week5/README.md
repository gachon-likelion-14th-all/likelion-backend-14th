Getter(getId): 객체의 필드(변수) 값을 읽어올 때 사용.
Setter(setId):객체의 필드(변수) 값을 변경하거나 저장할 때 사용

그냥 써도 되는데 굳이 쓰는 이유:
-setter를 쓰면 조건을 넣어서 원하는 값들만 넣을 수 있게 유도 가능
-데이터 구조 바껴도 외부 코드 영향 안 받음
-setter 안 만들고 getter만 만들면 그 값 자체는 수정 못하고 오직 읽기만 할 수 있게 만들 수 있음.

여기에선 필드가 private이기 때문에 그냥은 값을 못 빼 오기 때문에 getter 사용이 필수다.
